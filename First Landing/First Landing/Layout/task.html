Description

Let's start by setting the basic layout of your landing page.

First, you need to divide the HTML body into 3 parts: header, main, and footer. The header tag contains the navbar, the main tag comprises the content, and the footer tag, as you can guess, has the footer. You can write a placeholder text inside these tags like in the example to identify them.

Second, select the body and set the maximum height at 100% of the visible screen area. You can play with the line-height or font-family properties for the body.

Select the header, main, and footer and set their minimum height at 10%, 60%, 30% of the visible screen area, respectively. Additionally, you can give them a background color to set them apart.

Objectives

In this stage your website should have:

  • The header, main and footer tags in the HTML body with the order mentioned here;
  • Stick to the style rules for tags as described above.
